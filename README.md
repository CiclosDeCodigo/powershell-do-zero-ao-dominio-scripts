## Sobre este repositório

Este repositório contém os scripts, exemplos e automações apresentados no livro **PowerShell — Do Zero ao Domínio**, com foco em:

- Automação corporativa  
- Governança de servidores  
- Administração de Active Directory  
- Monitoramento e auditorias  
- Manipulação avançada de arquivos, logs e permissões  
- Integração entre PowerShell e Inteligência Artificial  

Todos os scripts seguem boas práticas e são documentados para uso seguro em ambientes reais.

---

## Estrutura do projeto

Os scripts estão organizados conforme os capítulos e as seções do livro.

Para os 50 scripts adicionais, eles estão agrupados por tema

| Módulo | Tema |
|--------|-------|
| A1–A10 | Manipulação de arquivos e análises |
| A11–A20 | Compartilhamentos, permissões e auditorias |
| A21–A30 | Métricas do sistema e Active Directory |
| A31–A40 | Serviços, jobs e automações |
| A41–A50 | Rede, diagnósticos e suporte |
| Extras | IA + PowerShell e integrações avançadas |

Os scripts serão adicionados gradualmente conforme revisão editorial.

Fornecer uma base sólida e progressiva para que alunos e profissionais:

- Apliquem Inteligência Artificial em tarefas operacionais e administrativas
